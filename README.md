# 7 Days to Die - FarmLife Polish Translation

To repozytorium zawiera kompletne i klimatyczne polskie tłumaczenie (Modlet) do modyfikacji FarmLife. Zmienia nazwy przedmiotow, opisy oraz interfejs stacji rzemieslniczych na jezyk polski.

## Instrukcja instalacji

Aby kod zadziałał poprawnie w Twojej grze, postępuj zgodnie z poniższymi krokami:

1. Pobierz to repozytorium klikając zielony przycisk **Code**, a następnie **Download ZIP**.
2. Wypakuj pobrane archiwum.
3. Skopiuj folder z tłumaczeniem **War3zuk FarmLife** (ten zawierający plik `ModInfo.xml`).
4. Wklej ten folder do katalogu `Mods` w głównym folderze Twojej gry *7 Days to Die*.
5. Uruchom grę – silnik gry automatycznie wczyta nowe pliki konfiguracyjne.
